    Introduction

    This plugin provides access to your channel PEAR library via Kodi.

    Installation

    1. Must have Kodi installed
    2. Must have a channel PEAR account with sources in your library
    3. Download the ZIP file
    4. Launch Kodi
    5. System → Settings → Add-ons → Install from zip file -> Navigate to the ZIP file

    Link your Library

    Once you've installed the plugin successfully, you'll need to link it to your channel PEAR library. Right click "channel PEAR" from your main Video Add-ons page in Kodi and then select "Add-on settings".

    Click the first button "Enter Kodi Channel URL from your Library" then paste or type in your custom Kodi URL. You can get your custom Kodi URL from your library at https://channelpear.com/library/ (must be logged in - click the "Kodi Channel" link at the top right of the page). Then select "Done".

    Then select the second button "After entering URL, click here to confirm". If you entered a valid URL, you will then be able to change the display name for your Library. Enter a new name and select "Done."

    Now you can click into the Add-on and watch compatible sources from your channel PEAR library!
